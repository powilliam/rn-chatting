# Chatting

Aplicativo desenvolvido para estudo sobre Offline First

## Start

Clonar esse repositório e executar o comando `yarn` e `yarn pod:install` para instalar as dependências.

## Rodar Projeto

- android: `yarn android`
- ios: `yarn ios`
